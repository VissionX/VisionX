# VisionX


VisionX is an assistive-technology project that provides real-time object detection, scene awareness, and audio descriptions to support visually impaired users. This documentation site is the central place for project setup, design decisions, API-like module references, and the team's development log.

## Project at a glance


- **Organization:** VissionX
- **Repository:** VisionX (https://github.com/VissionX/VisionX)
- **Language:** Python 3
- **AI / ML:** OpenCV, TensorFlow (YOLO planned)
- **Audio:** Vosk (offline), SpeechRecognition (optional)
- **Interface:** Kivy (planned)
